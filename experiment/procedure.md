<b>Lab Launching Steps: </b> <br>
1. Access the smart classroom application from the provided link or platform.<br>
2. Log in using the given credentials.<br>
3. Select the specific session or experiment assigned by the instructor.<br>
4. Ensure that all display systems in the classroom are powered on and connected to the
main control unit.<br>
5. Start the activity session from the instructor’s console to display questions on multiple
screens.<br><br>

<b>Working of the Experiment:</b><br>
1. Participants will view questions displayed within the virtual reality environment.<br>
2. Each participant will use their VR controller to select and submit answers.<br>
3. Multiple types of questions may be displayed.<br>
4. The system will collect and record responses for analysis.<br>
5. Participants can observe correct answers after each question.
